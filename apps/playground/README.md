# Fraud simulation sandbox (scenarios)

This folder holds **structured scenario fixtures** for demos and future split of a dedicated attack-testing UI.

The interactive playground currently ships inside **`apps/dashboard`** (Developer Playground tab). Use `scenarios/*.json` as canonical payloads for:

- Judge walkthroughs
- Scripted `scripts/simulate-fraud.ts` runs
- Future `apps/playground` Vite app (optional extraction)

## Scenarios

See `scenarios/` for JSON examples (SIM swap, high-value transfer, new device, clean baseline).
